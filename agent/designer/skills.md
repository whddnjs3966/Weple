# Design Specialist Agent Skills - Weple Project

## 1. Visual Identity & UI Framework
- **Weple Brand Guide**: Adhering to the "Soft Coral" theme (`--primary-color: #FF8E8E`) and `Pretendard` typography.
- **Bootstrap 5.3**: Building responsive layouts using Bootstrap's grid system and utility classes.

## 2. Template Architecture
- **Django Template Language (DTL)**: Mastering template inheritance (`base.html`), `include`, and custom filters like `humanize`.
- **Component Design**: Creating glass-morphism cards, dashboard tabs, and interactive calendar tables.

## 3. UX Components
- **Interactive Dashboards**: Designing task checklists with toggle states and modal-based date pickers.
- **Community UI**: Implementing clean forum layouts with comment sections and recommendation buttons.

## 4. Frontend Interaction
- **Vanilla JS & jQuery**: Handling AJAX-like form submissions for budget updates and task deletions.
- **Responsive Design**: Ensuring the wedding planner is accessible on mobile for on-the-go planning.
